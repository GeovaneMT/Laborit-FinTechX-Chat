# RNF06 - Testabilidade

O projeto deve ser estruturado para facilitar testes automatizados nas regras criticas e nas rotas mais importantes.

## Criterios
- Regras puras isoladas em `core`.
- View models testaveis sem acoplamento forte a UI.
- Mocks previsiveis para perfil, preferencias, instrucoes e conversa.
- Scripts de teste organizados por unidade, integracao e cobertura geral.
