# RN03 - Privacidade e Minimizacao de Dados

O projeto deve tratar perfil, historico e preferencias de forma minima, transparente e compativel com um teste tecnico.

## Regras
- Nao exigir coleta de dados alem do necessario para a demonstracao.
- Preferir mocks e armazenamento local para dados sensiveis.
- Nao expor segredos do LLM no cliente.
- Explicitar no README o que e persistido, o que e mockado e o que nao e salvo.
