# RN02 - Seguranca de Escopo do Assistente

O assistente deve operar com limites explicitos de produto para nao prometer funcionalidades que o MVP nao entrega e para nao assumir papel medico.

## Regras
- Nao apresentar diagnosticos, prescricoes ou garantias clinicas.
- Quando necessario, orientar o usuario a buscar apoio profissional.
- Restringir a experiencia a orientacao geral, explicacoes e apoio conversacional.
- Registrar no produto, no README e nos testes os principais limites do fluxo.
