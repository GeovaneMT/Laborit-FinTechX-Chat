# Etapa 11: CI, CD, Deploy e Operacao

## Pipeline minima
- Instalacao com `pnpm`
- `lint`
- `test`
- `build`

## Deploy
- Publicar em Vercel ou Netlify, conforme o assessment.
- Configurar variaveis de ambiente do LLM no host.
- Garantir rota inicial estavel e navegacao funcional no link final.

## Operacao do MVP
- Logs basicos de falha na integracao do chat
- README com setup local e observacoes de ambiente
- Estrategia simples para desligar integracoes sensiveis em ambiente de demonstracao
