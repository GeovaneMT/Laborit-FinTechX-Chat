# RF01 - Conversa com IA

O usuario deve conseguir abrir a rota de chat, digitar uma mensagem, enviar o prompt para a IA e visualizar o estado de processamento antes da resposta final.

## Criterios de aceitacao
- O campo de mensagem aceita texto curto e medio.
- O botao de envio so envia quando houver conteudo valido.
- O envio dispara uma Server Action ou adaptador equivalente do LLM.
- Durante a resposta, a interface mostra feedback de carregamento.
- O usuario pode enviar uma nova pergunta apos a conclusao da resposta.
