# RF04 - Instrucoes e Limites de Uso

A aplicacao deve ter uma rota dedicada para apresentar limites, capacidades e cuidados do assistente, alinhada a tela `Health Instructions`.

## Criterios de aceitacao
- A rota exibe orientacoes claras sobre memoria de contexto, limitacoes e possibilidade de erro.
- O conteudo e objetivo, humano e nao alarmista.
- O usuario consegue navegar dessa rota para o chat sem friccao.
- As orientacoes reforcam que a IA nao substitui atendimento profissional quando necessario.
