# RNF02 - Desempenho

O produto deve iniciar rapido, responder bem entre rotas e manter o chat fluido mesmo com respostas longas.

## Criterios
- `Splash` e `Onboarding` com carregamento leve.
- Chat com atualizacao otimista do estado local.
- Componentes e listas sem renderizacoes desnecessarias.
- Estrategia de cache e hidratacao coerente com o template Next.js 16 SPA.
