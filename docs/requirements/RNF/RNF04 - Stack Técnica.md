# RNF04 - Stack Tecnica

A stack do projeto deve ser compativel com o blueprint tecnico informado para o teste.

## Criterios
- Next.js 16, React 19 e TypeScript estrito.
- `pnpm`, Tailwind CSS v4, shadcn/ui, Radix UI e CVA.
- TanStack Query para leituras cliente, TanStack Form para formularios e Zustand para estado local/global pequeno.
- Vitest, Testing Library e MSW para qualidade.
