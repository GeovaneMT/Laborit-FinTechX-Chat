# RNF01 - Responsividade

A interface deve funcionar bem em mobile e desktop, preservando a hierarquia visual dos assets.

## Criterios
- Mobile-first como referencia principal.
- Largura maxima controlada para evitar esticar demais a UI no desktop.
- Navegacao, inputs e CTAs confortaveis para toque e mouse.
- Todas as 8 rotas devem manter usabilidade nas duas modalidades de tema.
